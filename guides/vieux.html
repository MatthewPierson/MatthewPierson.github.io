Vieux Guide

Requirements

  1. A macOS machine (hackintosh or legit mac)
  2. Python 3.5 or higher
  3. Copy of files from the Vieux repo

Instructions

  1. Run "pip3 install -r requirments.txt"
  2. Connect device to computer in DFU mode (NOT PWNDFU MODE)
  3. Run "./vieux -i path/to/.ipsw"
  4. Follow the programs instructions when needed

Done!
