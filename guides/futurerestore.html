Futurerestore Guide

Basic Information:

Futurerestore is a tool written by Tihmstar which is used to downgrade/upgrade 64/32 Bit iOS devices to unsigned iOS versions using previously saved SHSH(2) files.

The versions that you can downgrade to are limited by what SEP versions are currently signed, as the signed SEP must be compatible in order for the downgrade to work.
E.G iOS 12.4.4's SEP is compatible all the way down to 11.3 allowing for downgrades to 11.3 up to 12.4.4 for A7 and A8 devices that have 12.4.4's SEP signed.

Restores done with Futurerestore will typically be Untethered as they use a valid ApTicket for the restore, however some 32 Bit downgrades use invalid ApTickets or ApTickets from other devices, which mean the device will need to be put into PWNDFU mode and tether booted.

Requirements

  1. A macOS machine (hackintosh or legit mac), Linux machine or Windows Machine (Windows is NOT reccomended)
  2. Futurerestore binary from here or here (For Windows)
  3. SHSH with a generator (Check here for information on Generators)
  4. IPSW file for the iOS version you want to restore to (Download IPSW's from here)
  5. Knowledge of what SEP version is currently signed.

Instructions

  1. Ensure that your generator has been set to match the generator found in your SHSH2 file (Check here for information on Generators)
  2. Connect device to computer in either Normal mode or Recovery mode
  3. Run Futurerestore using one of the following methods:

    • If the latest signed SEP version is compatible with what iOS version you want to restore to then run: "./futurerestore -t path/to/shsh --latest-sep --latest-baseband path/to/.ipsw"
    • If you want to use an older but still signed SEP then run:"./futurerestore -t path/to/shsh -s path/to/sep.im4p -m path/to/buildmanifest.plist -b path/to/baseband.bbfw -p path/to/buildmanifest.plist path/to/.ipsw"


  4. Wait for the restore to finish
  5. If you run into errors follow this and if that doesn't solve your issues then google the error code.

Done!
