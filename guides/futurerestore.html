Futurerestore Guide

Requirements

  1. A macOS machine (hackintosh or legit mac) or Linux machine
  2. Futurerestore binary from here
  3. SHSH2 with a generator

Instructions

  1. Ensure that your generator has been set to match the generator found in your SHSH2 file (this can be done using various methods)
  2. Connect device to computer in either Normal mode or Recovery mode
  3. Run "./futurerestore -t path/to/shsh --latest-sep --latest-baseband path/to/.ipsw"
  4. Wait for the restore to finish

Done!
